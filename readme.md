# C-Pirate

Small IDE which combines the simplification of using a terminal as an editor and using buttons to compile and run.

## Installation
To use this you need GTK3+

```bash
sudo apt-get install libgtk-3-dev
```

## Give permissions
Use +x or 777

```bash
chmod +x run.sh
```


## Start
To run the script

```bash
./run.sh
```
